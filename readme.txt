* Shortcode Enabled

<strong> How to use this plugin</strong>

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>AA Sitemap</strong>" activate it.<br />
